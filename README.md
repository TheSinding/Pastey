# pastey
A simple tool for pasting to https://pbpt.pw


##### Dependencies: 
+ curl
+ sed

##### Usage:
pastey [OPTION] [FILE]   
-h Shows help   
-f Upload file   

##### Installation: 
`git clone https://github.com/TheSinding/pastey`   
`cd pastey`   
`chmod +x pastey`   
`cp pastey /usr/local/bin/`
